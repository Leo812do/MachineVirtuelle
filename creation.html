Création de notre Machine Virtuelle

Cette partie présente notre creation d'une machine virtuelle Ubuntu à l’aide de captures d’écran réalisées dans VirtualBox.

Écran de connexion de la machine virtuelle Ubuntu avec les différents utilisateurs. Le compte d'admin, les comptes des deux binômes et le compte de Guo.

Bureau Ubuntu après le démarrage de la machine virtuelle.

Utilisation du terminal pour demander une liste des applications dans Ubuntu ou dans le compte en utilisant la commande :
sudo ufw app list.

Utilisation de la commande : sudo ufw status pour vérifier les applications si ils sont actives ou disponibles.

Ce commande : sudo systemctl status apache2 permet de voir en détail l'information et l'etat de l'application Apache2.
