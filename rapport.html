Organisation et structure du site

Le site web a été conçu avec une structure simple et claire afin de faciliter la navigation et la compréhension des informations. Il est composé de plusieurs pages HTML, chacune correspondant à une partie précise du projet.

Les balises sémantiques header, main et footer ont été utilisées afin de respecter les bonnes pratiques du développement web et de rendre le code plus lisible et plus facile à maintenir.

Utilisation du HTML

Le HTML permet de structurer le contenu du site. Le header contient le titre principal et le menu de navigation, le main regroupe le contenu principal de la page, et le footer affiche les informations générales du projet.

Les informations sont organisées à l’aide de titres et de paragraphes afin d’améliorer la lisibilité et de hiérarchiser les contenus.

Utilisation du CSS et de Bootstrap

Bootstrap est utilisé pour assurer une mise en page responsive et structurée. Il permet au site de s’adapter automatiquement aux différentes tailles d’écran, comme les ordinateurs, les tablettes et les smartphones.

Un fichier CSS personnalisé est utilisé en complément de Bootstrap afin de modifier le style du site, notamment les couleurs, la typographie et les effets visuels. Cette combinaison permet d’obtenir un site à la fois moderne, lisible et cohérent avec le thème du projet.

Positionnement du pied de page

Le pied de page est positionné en bas de la page grâce à l’utilisation du Flexbox sur la balise body. Le contenu principal occupe automatiquement l’espace disponible, ce qui garantit un affichage propre, même lorsque la page contient peu de contenu.
